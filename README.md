# multiroom-chat-realtime

This project is a multiroom chat developed to practice real-time, bidirectional and event-based communication with Node.js.

## Getting Started 

``` bash
# install dependencies
npm install

# serve at localhost:8080
node app.js
# or install nodemon and run nodemon app.js
```
## Built With

* [Express](https://www.npmjs.com/package/express) - The web framework used.
* [Consign](https://www.npmjs.com/package/consign) - Used to autoload models, routes,configs and controllers.
* [Socket.IO](https://socket.io/docs/) - Socket.IO enables real-time bidirectional event-based communication.
* [EJS](https://www.npmjs.com/package/ejs) - Front-end - Embedded JavaScript templates.
